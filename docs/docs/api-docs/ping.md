# Ping

These are the docs for the ping endpoint, used for checking if the PasteMyst instance is up.

## Ping

Pings PasteMyst.

<details>
    <summary><code>GET /api/v3/ping</code></summary>

    ##### Response

    Status `200 OK` if it's up and running.
</details>
